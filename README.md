# Ai_project
## Overview
This project is an AI-based recruitment system that includes:
- ATS (Applicant Tracking System)
- Candidate Screening
- Interview Analysis
- Scoring System

## Project Structure
- data/
- parsers/
- ats_engine/
- screening_ai/
- interview_ai/
- scoring/
- utils/
- tests/

## Setup
1. Create virtual environment
2. Install dependencies:
   pip install -r requirements.txt

## Run
python main.py
